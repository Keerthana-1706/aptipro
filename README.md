# AptiPro

AptiPro is a web-based application designed to conduct online aptitude tests in a secure and structured environment. The platform allows users to take assessments and receive instant performance feedback.

## Features

* User authentication
* Online aptitude test interface
* Timed assessments
* Automatic evaluation and score display
* Performance tracking
* Simple and user-friendly interface

## Tech Stack

* HTML
* CSS
* JavaScript
* Node.js

## Project Structure

```
aptipro
│
├── code        # Main application files
├── db          # Data files
└── .vscode     # Editor configuration
```

## How to Run

1. Clone or download the repository.
2. Open the project folder.
3. Run the application using your preferred development environment or local server.
